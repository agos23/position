1.¿Cuál es la diferencia principal entre position relative y position absolute?

La principal diferencia entre position relative y position absolute es que el position relative permite desplaszar un elemento desde su posicion original mientras que el position absolute coloca a un elemento en una posicion especifica.

2.¿Qué valor de position se utiliza por defecto para los elementos HTML?

Por defecto para los elementos HTML se utiliza las propiedades top, bottom, left o right.

3.¿Qué propiedad se utiliza para establecer la capa de apilamiento de elementos superpuestos?

La propiedad que se utiliza para establecer la capa de apilamiento es z-index.

4.¿Cuándo se utiliza position fixed en CSS?

Position fixed se utiliza para fijar un elemento.

5.¿Cómo se comporta un elemento con position sticky al desplazarse?

un elemento se comporta como "pegajoso" en relación a su contenedor o al área de visualización del navegador.

6.¿Cuál es el valor de position adecuado para un elemento que se desea que siga el flujo normal del documento?

El valor de position adecuado seria posittion estatic.

7.¿Cuál es la forma más común de mover un elemento con position relative?

utilizando las propiedades top, bottom, left o right.
